>>=|XMLHttpRequest (XHR)|=>>
    XMLHttpRequest ek JavaScript object hai jo AJAX ke through server se asynchronously data fetch ya send karta hai, bina page reload kiye. Naam me XML hai lekin aaj ke time me mostly JSON use hota hai.
    -> JavaScript built-in object -> server se data lana ya bhejna page reload kiye bina.
    -> AJAX – Asynchronous JavaScript And XML

>>=|XHR ka basic life cycle (MOST IMPORTANT)|=>>
    1. Object create
    2. Request configure
    3. Request send
    4. Response receive
    5. Data Use

>>=|Steps|=>
    Step 1 – XHR object banana
        const xhr = new XMLHttpRequest();
    Step 2 – Request open karna
        xhr.open("GET", "https://api.example.com/image", false); -> by default true
    Step 3 – Response ka type set karna
        xhr.responseType = "json";
            +---------------------------+
            | Type       | Use          |
            | ---------- | ------------ |
            | "text"     | Normal text  |
            | "json"     | JSON data    |
            | "blob"     | Image / file |
            | "document" | XML / HTML   |
            +---------------------------+
    Step 4 – Event Listener lagana
        xhr.addEventListener("load", () => {
          console.log(xhr.response);
        });
